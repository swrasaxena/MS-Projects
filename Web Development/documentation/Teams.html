Teams Page


On the top of the Teams page is an image which I took from google images.
Next is a table with NFL team's information populated. I have used ESPN API to get the following data in the table:
  • Team ID
  • Team Name
  • Location
  • External link to team profile

I have used the background color for each row of team's information to be the color of the team retrieved from the API. I am using javascript and jQuery's ajax() function to retrieve data and populate it in table format.
